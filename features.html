Core Features

Multi‑segment Engine

Mix segments (Asphere, Bevel, Flat) with per‑segment parameters and lengths. Designed to match real lens layouts.

Programmable Tool Radius

Explicit tool radius in generation logic. Clear filenames like Asphere_R0.250_M3.

Per‑side Corrections

Left / Right / Average application from MOD files with percent scaling and X‑shift.

SAPR

Smart Adaptive Profile Reconstruction: restore ideal profile, subtract measured, keep only true error.

Chamfer & Retract

Automatic retract after chamfer finish to prevent tool edge hits.

Visualization

2D profile preview with segment labels and lengths — see before you cut.
