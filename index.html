About

‚ú¶ DTM Studio ‚Äî Diamond Turning Manager

DTM Studio is software developed in Riga by an SPDT machine operator, built specifically for operators. It addresses the everyday workflow on Precitech and Nanotech machines and eliminates the routine steps where most time is wasted or mistakes often occur.

Instead of manually editing code or switching between multiple utilities, DTM brings everything into one place.

What you can do in DTM Studio:

  • Generate G-code for Precitech (.PGM) and Nanotech (.NC).
  • Work with profile corrections: load MOD files, select side, scale by percentage, and stack multiple corrections.
  • Tool radius support: included directly in trajectory calculations.
  • Flexible project structure: stores form, corrections, and parameters together.
  • Optimus LDT support: hybrid cutting code ‚Äî standard and laser-enabled.
  • Log and error control: tracks every step, reducing risk of mistakes.
