About

✦ DTM Studio — Diamond Turning Manager

DTM Studio is software developed in Riga by an SPDT machine operator, built specifically for operators. It addresses the everyday workflow on Precitech and Nanotech machines and eliminates the routine steps where most time is wasted or mistakes often occur.

Instead of manually editing code or switching between multiple utilities, DTM brings everything into one place.

What you can do in DTM Studio:

  • Generate G-code for specific machines: Precitech (.PGM) and Nanotech (.NC).
  • Work with profile corrections (MOD files): load, select side (Left/Right/Average), scale by percentage, and stack multiple corrections.
  • Tool radius support: programmable Tool Radius is included directly in trajectory calculations, preventing double compensation and ensuring accuracy.
  • Flexible project structure: when saving, the project stores form, corrections, and applied parameters, so work can be restored without re-entering everything manually.
  • Log and error control: the operator can track every preparation step, reducing the risk of using incorrect parameters.
  • Simple file organization: G-codes, corrections, and project files are saved together in one folder — no need to search across directories.

Why this matters for operators:

  • Speeds up program preparation: no need to hand-edit commands or save in different formats.
  • Minimizes risk of scrap due to human error: corrections are applied precisely, and trajectories are validated before loading to the machine.
  • Provides a clear, unified workflow: from profile loading to ready-to-cut code.

DTM Studio is not a concept or a “future update.” It is already a working tool that simplifies daily SPDT programming and makes preparation faster, safer, and more reliable.
