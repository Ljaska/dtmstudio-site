Core Features

G-code Generation

Create ready-to-cut code for Precitech (.PGM) and Nanotech (.NC) machines without manual edits.

Profile Corrections

Load MOD files, choose side (Left / Right / Average), scale by percentage, and apply with precision.

Corrections Stack with Undo

Safely stack multiple corrections, prevent duplicates, and revert steps at any time.

Tool Radius Mode

Programmable tool radius built directly into trajectory calculations, avoiding double compensation.

Save Order

One-click save: project, G-codes, and corrections are stored together in a single folder.

Correct and Go

Fast correction cycle — apply measured data, regenerate, and cut again without extra steps.

Loop File One-Click

Create programs for all turning sides (with or without bevels) plus ready loop file in a single action.

Surface + Diameter Turning

Simplified workflow for cutting both profile and cylinder in one setup.

Error Prevention

Built-in checks minimize operator mistakes and ensure correct program parameters.

Speed

Preparation time reduced several times compared to manual code editing.

Simplified Geometry Presets

Quick generation of Sphere, Asphere, Convex, and Concave surfaces without manual formulas.

Optimus LDT Support

Full compatibility with Micro-LAM Optimus laser-assisted diamond turning. Generate hybrid programs — standard and laser-enabled — with no extra steps.
